大模型的七十二变 - 大模型有哪些格式

发表于 2025-03-25 更新于 2025-05-12 分类于 技术

大模型有不同的格式. 如果你最初接触的是 meta 家的羊驼大模型 llama, 那么 GGUF 格式你肯定再熟悉不过.

这是由著名的开源推理项目 llama.cpp 研发的格式, 受到主流的推理项目如:llama.cpp、ollama和vLLM 的支持.

如果你是用 HF 家推出的 transformers框架训练, 那么 safetensors 一定是你打交道最多的.它由 Hugging Face 开发的safetensors解决了传统 Python 序列化方法 (如 PyTorch 使用的 pickle) 中存在的安全性和效率问题。该格式使用受限的反序列化过程来防止代码执行漏洞。 那么一共有哪些模型格式呢, 各自有哪些特点呢?

下面来逐一比较.
