Lora微调是什么

发表于 2025-03-23 分类于 技术

Lora微调是什么

Lora微调(Low-Rank Adaptation)，是一种高效率的微调技术。可以降低微调参数的数量，从而降低微调所需要的资源。

Lora的论文地址：https://arxiv.org/abs/2106.09685

Lora微调为什么高效？

让我们来看下Lora的原理。

对于全参微调的过程，可以简单表示

W1= W + △W

W是微调前的权重参数，△W是权重的更新，W + △W得到新的权重参数W１。需要注意，△W在微调的过程中更新，和W具有一样的维度大小。

Low-Rank Adaptation，Rank是矩阵的秩，秩是矩阵中线性无关的向量值，不会超过最小的维度值。比如一个 30* 2的矩阵，rank 最大为2.

降低秩，某种程度上可以减少训练权重。

Lora做了一个这样的事情。将 △W 近似成 AB, 两个矩阵相乘.

W1≈ W + AB

A和B是一个秩相对很小的矩阵, 一般设置为16.

你会发现了吗？ 将AB替换 △W之后,需要训练的参数就大大减少.

从工程实践的角度来看, Lora参数可以单独训练, 单独保存, 推理的时候再和原始的权重一起加载.

这个得益于矩阵乘法乘法的分配律,

XW1 = XW + XAB

X为模型的输入，求解输出的时候，我们可以分开计算原模型的输出XW 和Lora模型的输出XAB，再把他们拼接起来。

以下是以LLM学习新手的视角列举的24个关于线性代数的高频问题，并站在老师的角度给出简明解答：

一、矩阵与向量基础

  1. 什么是矩阵，为什么LLM中到处都有矩阵？
    • 矩阵是二维数组，用于存储数据或表示变换。在LLM中，矩阵用于存储权重、输入和输出的关系。
  2. 什么是向量，向量和矩阵有什么区别？
    • 向量是只有一行或一列的矩阵，表示单个数据点或特征集合。矩阵可以看作是多个向量的集合。
  3. 矩阵乘法为什么重要？
    • 矩阵乘法是LLM中计算输入与权重关系的核心操作，用于前向传播和反向传播。
  4. 矩阵转置有什么用？
    • 矩阵转置用于调整矩阵形状，特别是在计算点积或反向传播时。
  5. 什么是单位矩阵，为什么它重要？
    • 单位矩阵是对角线全为1，其余为0的矩阵。它是矩阵运算中的“1”，不会改变矩阵的值。

二、子空间与维度

  1. 什么是列空间和零空间？
    • 列空间是矩阵列的线性组合，表示数据的有效维度；零空间是矩阵映射到零的向量集合，用于判断解的存在性。
  2. 什么是线性相关性？为什么要关注它？
    • 如果一个向量可以由其他向量线性组合得到，它们是线性相关的。在LLM中，线性无关性表示特征独立性。
  3. 什么是基？基和维数有什么关系？
    • 基是一个向量集合，可以表示空间中的所有向量。基的数量就是维数。
  4. 为什么LLM中要关注高维空间？
    • 高维空间能表示更多的特征和复杂关系，但也带来计算和过拟合的挑战。
  5. 正交向量是什么意思？
    • 正交向量是互相垂直的向量，表示它们之间没有相互影响。

三、矩阵分解

  1. 什么是特征值和特征向量？为什么重要？
    • 特征值表示矩阵变换的缩放因子，特征向量表示不变方向。在LLM中，它们用于降维和稳定性分析。
  2. 什么是奇异值分解（SVD）？
    • SVD将矩阵分解为三个部分，用于降维、压缩和特征提取。
  3. 对角化是什么意思？为什么要对角化矩阵？
    • 对角化将矩阵转换为对角矩阵，简化运算，特别是在动力学分析中。
  4. 什么是QR分解？
    • QR分解将矩阵分解为正交矩阵Q和上三角矩阵R，用于求解线性方程。
  5. LU分解在LLM中有用吗？
    • LU分解将矩阵分解为上三角和下三角矩阵，用于高效求解方程组。

四、投影与最小二乘

  1. 什么是投影矩阵？
    • 投影矩阵用于将数据投影到一个子空间，常用于降维和回归分析。
  2. 最小二乘法是如何工作的？
    • 最小二乘法通过最小化误差平方和，找到最佳拟合线或平面。
  3. 什么是Gram-Schmidt正交化？
    • 它是一种将向量集合转化为正交基的算法，用于提高数值稳定性。

五、优化与正定性

  1. 什么是正定矩阵？为什么它重要？
    • 正定矩阵的特征值全为正，表示优化问题有唯一解，常用于梯度下降。
  2. 什么是半正定矩阵？
    • 半正定矩阵的特征值非负，用于核方法和支持向量机。
  3. 矩阵秩是什么？为什么它重要？
    • 秩是矩阵中线性无关行或列的数量，表示数据的有效维度。

六、数据变换与应用

  1. 什么是线性变换？

    • 线性变换是通过矩阵操作改变数据的表示方式，用于特征提取。

  2. 基变换在LLM中有什么用？

    • 基变换用于切换数据的表示空间，特别是在降维和特征工程中。

  3. 伪逆矩阵是什么？为什么需要它？

    伪逆是非方阵的广义逆，用于解决欠定或超定问题。

对核心概念和关键问题建立整体的框架即可，遇到细节问题再即用即查。

    矩阵乘法是神经网络前向传播和反向传播的核心操作；逆矩阵在优化问题中有时会用到。

2. 矩阵分解与空间

  • 第06集 列空间和零空间
    列空间和零空间帮助理解矩阵的性质，尤其在数据降维和特征提取中。
  • 第10集 四个基本子空间
    四个子空间（列空间、零空间、行空间、左零空间）是矩阵运算和数据表示的核心。
  • 第16集 投影矩阵和最小二乘
    最小二乘法是机器学习中线性回归的理论基础。
  • 第17集 正交矩阵和Gram-Schmidt正交化
    正交化在构造正交基和特征值分解中非常重要。
  • 第21集 特征值和特征向量
    是降维（如PCA）、谱聚类和神经网络稳定性分析的核心。
  • 第22集 对角化和A的幂
    对角化用于简化矩阵运算，尤其在动力系统和递归网络中。
  • 第30集 奇异值分解（SVD）
    SVD是推荐系统、图像压缩和自然语言处理（如词嵌入）中的关键技术。

3. 优化与正定性

  • 第26集 对称矩阵及正定性
    对称正定矩阵在优化问题（如梯度下降）中至关重要。
  • 第28集 正定矩阵和最小值
    正定性保证优化问题有唯一解，是深度学习中损失函数设计的重要理论。

4. 数据变换与应用

  • 第31集 线性变换及对应矩阵
    线性变换是深度学习中激活函数和特征变换的数学基础。
  • 第32集 基变换和图像压缩
    基变换用于数据表示的变换，图像压缩相关技术（如PCA）也依赖于此。
  • 第34集 左右逆和伪逆
    伪逆矩阵在处理欠定或超定系统时非常重要，广泛用于数据拟合和模型校正。

5. 其他重要知识点

  • 第14集 正交向量与子空间
    正交性是高效计算和简化问题的关键。
  • 第23集 微分方程和exp(At)
    在时间序列建模和连续时间系统中有应用。
  • 第24集 马尔可夫矩阵、傅立叶级数
    马尔可夫矩阵用于概率建模，傅立叶变换在信号处理与卷积神经网络中广泛应用。

总结

最重要的知识点包括：

  • 矩阵分解（LU分解、SVD、特征值分解）
  • 空间与子空间（列空间、零空间、正交性）
  • 投影与最小二乘法
  • 正定矩阵与优化
  • 奇异值分解与基变换

这些内容构成了大模型数学基础的核心部分。
